$_SERVER：包含服务器及其执行环境的信息，如头信息、路径等。
$_GET：包含通过 HTTP GET 方法传递给脚本的变量。
$_POST：包含通过 HTTP POST 方法传递给脚本的变量。
$_COOKIE：包含通过 HTTP Cookies 传递给脚本的变量。
$_FILES：包含通过 HTTP POST 方法上传的文件信息。
$_ENV：包含环境变量。
$_REQUEST：包含 $_GET、$_POST 和 $_COOKIE 的内容。请注意，由于它会合并其他超全局变量的数据，因此不建议在实际代码中使用 $_REQUEST，因为它可能导致安全隐患。
$_SESSION：包含当前脚本可用的会话变量。在使用 session_start() 函数启动会话后，可以使用 $_SESSION 来存储和检索跨页面请求的数据。
$GLOBALS：包含所有全局变量的关联数组
