    一种曲线救国

    直接用Box打

    这种内网速度是最快的, 但是和web端交互是非常慢的, 而且工具也不全

    代理

    OpenVPN代理经常卡

    开box来做个代理

    wget VPS/chisel_amd_linux && chmod +x chisel_amd_linux && ./chisel_amd_linux client VPS:7000 R:0.0.0.0:7777:socks

    VPS上./chisel server -p 7000 --reverse

    这样速度能快很多,但是还是建议买个支持UDP的好点的梯子

    有前置代理的OpenVPN速度是最快的

    前置代理

    OpenVPN

    正常来说修改ovpn文件

    socks-proxy 127.0.0.1 7890

    但是如果你使用的是MacOS的话, 用surge代理软件

    开个全局代理, 增强接管一下, 那样就可以直接使用了, 虚拟机里也不用再多做设置了

    另外同时使用surge和proxifier的时候需要注意一下DNS循环解析
